# SmartERP

SmartERP is a Tally-inspired Enterprise Resource Planning (ERP) web application.

## Features

- Company Management
- Ledger Management
- Inventory Management
- Voucher Entry
- Reports
- Balance Sheet
- Banking
- Billing
- PDF Export
- Keyboard Shortcuts

## Technology Stack

- React.js
- Bootstrap
- jsPDF
- Supabase (Upcoming)
- Vercel Deployment
